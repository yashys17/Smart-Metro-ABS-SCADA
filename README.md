# 🚇 Smart Metro Automatic Block Signalling & SCADA Monitoring System

## 📌 Project Overview

The **Smart Metro Automatic Block Signalling & SCADA Monitoring System** is a web-based simulation of a metro Operations Control Centre (OCC).

The project demonstrates how metro railway systems monitor train movement, control signals, detect block occupancy, and handle emergency situations using **Automatic Block Signalling (ABS)** and **SCADA concepts**.

The system provides a real-time dashboard for monitoring trains, signals, blocks, alarms, and overall system status.

---

# 🎯 Project Objectives

- To understand the working principle of Automatic Block Signalling.
- To simulate metro train movement and safety control.
- To demonstrate signal-based train protection.
- To develop an OCC-style SCADA monitoring interface.
- To understand railway signalling and control concepts.

# ⚙️ Features

## 🚆 Train Monitoring

- Real-time monitoring of UP and DOWN trains.
- Displays train position and current block.
- Simulates multiple train movements.

## 🚦 Automatic Block Signalling

The system follows railway signal principles:

🟢 **GREEN Signal**
- Track route is clear.
- Train can move normally.

🟡 **YELLOW Signal**
- Caution condition.
- Train reduces speed.

🔴 **RED Signal**
- Next block is occupied.
- Train must stop.

---

## 📦 Block Occupancy Detection

- Railway track is divided into multiple blocks.
- Each block is monitored as FREE or OCCUPIED.
- Prevents unsafe train movement.

# 📊 What is SCADA?

**SCADA stands for Supervisory Control And Data Acquisition.**

SCADA is an industrial monitoring and control system used to collect real-time data from field devices and display it to operators.

In this project, SCADA is used for:

- Monitoring train positions
- Displaying signal conditions
- Tracking block occupancy
- Managing alarms
- Providing an operator interface
